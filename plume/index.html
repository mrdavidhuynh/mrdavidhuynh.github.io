Plume WiFi

—

Plume's mission is to reinvent home WiFi with their adaptive WiFi technology. Their product opened up to pre-orders in the summer of 2016.

Before their official launch, I worked with the Plume team designing portions of the iOS app used to setup and manage the hardware.

My Role
—

  • 01 Setup Flow UX/UI
  • 02 Troubleshooting UI
  • 03 Illustrations & Visual Design

Setup flow
—

The onboarding that I worked on involved teaching the user how to set up the hardware, connect it to the internet, and create their WiFi network.

I designed these screens with a lot of illustrations and animations to visually guide the user through this process.

Troubleshooting Guides
—

I was also tasked with designing screens that guide the user through connection or hardware problems that they might encounter.

These screens highlight the possible problem while offering some possible solutions.
